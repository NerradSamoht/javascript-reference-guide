Variables

Declaring a variable var myName;

If no value is assigned JavaScript assigns a value of undefined.

To assign a value use the = operator.

var a;
a = 1;

A variable can be assigned a value in the same line var a = 1;.

It is possible to create a variable without using var.

myString = "A global variable."

This however will create a global variable, which could lead to some unexpected results, so always best to use var.

Strings

Concatenating strings

To concatenate a string you can use the + operator.

var a = "Hello";
var b = "World";
var c = a + " " + b; // Hello World

To concatentate a string to the end of an existing string you can use the += operator.

var a = "Hello"
a += " World"; // Hello World

Escaping strings

Strings values can be written using single or double quotes. You should always start and end with the same type of quote, if using quotation marks inside a string then you can use the alternate quote. For example:

var str = "I'm a string";

Quotation marks can also be used by escaped, this is done using a backslash.

var str = "I\"m a string";

Built in String properties

string.length
var name = "Hubert J. Farnsworth";
name.length; // 20
string[index]

JavaScript uses zero-based indexing, so when counting, everyting starts at 0. To return a character at a specific index, use bracket notation and the index number.

var name = "Hubert J. Farnsworth";
name[5]; // t

To find the second to last character of a string, you can use the length property and a subtraction operator inside the bracket notation.

var name = "Hubert J. Farnsworth";
name[name.length - 2]; // t

Arrays

Arrays are used to store several pieces of data in one place. Data can be of any type, string, number, object, etc.

var myArray = ["pizza", "chips", "burgers"];

You can even have an array of arrays, this is called a know as multi-dimensional array

var myArray = [["pizza","chips", "burgers"], [1, 10, 8], ["sandwich", 4]];

To access data in arrays use bracket notation and indexes.

var array = [1,2,3,6,5];
array[2]; // 3

To return data from a multi-dimensional array

var array = [[1,2,3], ["apple","orange","pear"]];
array[1][1]; // orange

Changing a value of an array

var array = [1,2,3,6,5];
array[3] = 4;

To append an item to the end of an array use the push() function

array.push(6); // array is now 1,2,3,4,5,6

To remove an item from the end of an array use the pop() function

array.pop(); // array is now 1,2,3,4,5

To remove an item at the start of an array use the shift() function

array.shift(); // array is now 2,3,4,5

To append an item to the start of an array use the unshift() function

array.unshift(1); // array is now 1,2,3,4,5

Functions

An example of a function

function functionName() {
  console.log("Hello World");
}

An example of a function stored in a variable

var functionName = function() {
  console.log("Hello World");
}

To invoke a function functionName();

An example of a function with parameters:

function sum(a, b) {
  console.log(a + b);
}
sum(2, 4); // Outputs 6

A returned value for a function can be stored in a variable

var answer = sum(5, 5); // answer is 10
